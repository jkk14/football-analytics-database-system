# ⚽ Football Analytics - Relational Database Management System

A relational database system developed for football analytics, player performance monitoring, injury tracking, training load management, and transfer decision support.

## 📌 Project Overview

This project demonstrates how relational database systems can be used to manage football-related data and support decision-making through SQL analytics and business intelligence reporting.

## 🛠️ Technologies Used

- MySQL Workbench
- SQL
- Power BI
- Entity Relationship Modelling (ERD)

## 📂 Project Files

- Database Script (.sql)
- ERD Diagram
- Power BI Dashboard Screenshot
- Project Report

## 🏗️ Database Entities

- Players
- Matches
- PlayerStats
- TrainingLoad
- Injuries
- TransferTargets
- TransferDecisions
- Clubs

## 📊 Key Features

- Relational database design and normalization
- Primary and foreign key relationships
- SQL queries and joins
- Views and stored procedures
- Query optimization using indexes
- Power BI dashboard reporting
